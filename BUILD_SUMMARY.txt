🚀 QUICK START:

1. Go to backend directory:
   cd backend

2. Install dependencies:
   npm install

3. Configure environment:
   cp .env.example .env
   # Edit .env with your MongoDB URI

4. Start server:
   npm run dev

5. Test health:
   curl http://localhost:5000/api/health

6. Seed database (optional):
   node seed.js

7. Test endpoints:
   curl http://localhost:5000/api/projects
