================================================================================
                            API INTEGRATION DETAILS
================================================================================

CONTACT FORM FLOW:
  1. User fills form (ContactSection.jsx)
  2. Validation check (frontend + backend)
  3. POST to http://localhost:5000/api/messages
  4. Server validates:
     • Name: 2-100 characters
     • Email: Valid email format
     • Message: 10-1000 characters
     • Rate limit: 5 per email per hour
  5. Store in MongoDB
  6. Return response (201 success, 400/429 error)
  7. Display success/error message in UI

FEATURES:
  ✅ Form validation (frontend + backend)
  ✅ Rate limiting (5 msgs/hour/email)
  ✅ Loading state with spinner
  ✅ Success message
  ✅ Error messages (validation, rate limit, server)
  ✅ Form clearing after submission
  ✅ Database persistence
  ✅ Beautiful UI maintained
